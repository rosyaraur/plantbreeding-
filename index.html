Welcome to plantbreeding package project

The R-package plantbreeding collection of functions for analysis of data and visualization of results from Plant Breeding and Genetics experiments. Different functionalities of this project include different aspects of data analysis plant breeding specialized experiments and designs such as Planning and Analysis of Plant breeding specialized designs, Estimation of variance components, heritability, and genetic correlation, Stability and genotype x environment interaction analysis Diversity and variability analysis, Selection and Selection index and some plotting features.

One of the main reasons for developing this package is that develop functions that are interest to plant breeders and geneticists. Your feedback and suggestions are welcome.

Installation and Download

To install the latest development version of package plant breeding from R-Forge (if you are running a recent R version), use

install.packages("plantbreeding",repos="http://r-forge.r-project.org")

The program is also available here at Github.

Current version of plantbreeding is 1.1.1

Note: The current version of plantbreeding has fewer functions than 1.1.0 but do not depend upon any other packages. PDF manual pdf version of plantbreeding manual can be available on request.
